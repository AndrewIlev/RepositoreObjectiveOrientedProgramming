## Опис завдання

### OOP-завдання
Реалізуйте систему рецептів:

1. Абстрактний клас `Dish` з атрибутами `name`, `cooking_time_min: int` та абстрактним методом `get_nutrition() -> dict`.
2. Класи `VeganDish` та `MeatDish` що успадковують `Dish` та реалізують `get_nutrition()`:
   - `VeganDish` має атрибут `calories: int` — повертає `{"calories": ..., "type": "vegan"}`
   - `MeatDish` має атрибути `calories: int` та `protein_g: float` — повертає `{"calories": ..., "protein_g": ..., "type": "meat"}`
3. Клас `CookBook` що:
   - зберігає рецепти у приватному словнику `__recipes: dict[str, Dish]` (інкапсуляція)
   - методи `add(dish)`, `find(name: str) -> Dish | None` та `list_all()` (поліморфізм через `get_nutrition()`)

### AI-агент
- **Інструмент (tool):** функція `get_recipe_info(dish_name: str) -> dict` — **всередині функції** створює об'єкт `CookBook`, наповнює його заздалегідь визначеними стравами `VeganDish` та `MeatDish`, викликає `find(dish_name)`, отримує харчову цінність через `get_nutrition()` та повертає її разом із `cooking_time_min`. Якщо страва не знайдена — повертає `{"found": False}`.
- **Промпт агента:** агент є кулінарним помічником. Він надає інформацію про рецепти, калорійність та тип страви, а також дає рекомендації щодо харчування. Відповідає **українською мовою**.
- **Демонстрація:** поставте агенту 3 запитання про різні страви.
